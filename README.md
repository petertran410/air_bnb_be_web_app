# Phạm Đức Đạt

1. USERS
-   GET danh sách users
     -  http://localhost:8080/api/users
-   POST tạo mới users
     -  http://localhost:8080/api/users/new
-   PUT update users
     -  http://localhost:8080/api/users/update/1
-   DELETE users
